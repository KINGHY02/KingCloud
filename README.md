# KingCloud

KingCloud 是基于ChromeGo开发的一个多代理工具整合客户端界面，基于 Electron 框架开发，支持多种代理工具的管理和使用。

## 功能特性

- **多工具支持**：集成了 clash.meta、Xray、hysteria、hysteria2、juicity、mieru、naiveproxy、shadowquic、singbox 等多种代理工具
- **IP 更新**：支持一键更新 IP 地址，获取最新的代理配置
- **工具管理**：可视化管理所有代理工具的启动、停止和状态监控
- **浏览器集成**：一键打开配置好代理的浏览器，快速访问网络
- **现代化界面**：采用现代化的 UI 设计，支持浅色/深色主题切换
- **响应式布局**：适配不同屏幕尺寸，提供良好的用户体验

## 系统要求

- Windows 10/11
- Node.js 16.0 或更高版本
- npm 8.0 或更高版本

## 快速开始

### 方法一：使用启动脚本

1. 双击 `start.bat` 文件
2. 脚本会自动检查并安装依赖
3. 等待应用启动

### 方法二：手动安装

1. 克隆或下载项目到本地
2. 打开命令行，进入项目目录
3. 安装依赖：
   ```bash
   npm install
   ```
4. 启动应用：
   ```bash
   npm start
   ```

## 使用方法

1. **选择工具**：从左侧边栏选择一个代理工具
2. **更新 IP**：点击"IP 更新"区域的按钮选择 IP 源
3. **启动代理**：点击"启动"按钮启动代理服务
4. **打开浏览器**：点击"打开浏览器"按钮访问网络
5. **停止代理**：使用完毕后点击"停止"按钮停止服务

## 常见问题

### 1. 启动失败怎么办？
- 检查 Node.js 和 npm 是否正确安装
- 检查网络连接是否正常
- 查看命令行窗口的错误信息

### 2. IP 更新失败怎么办？
- 检查网络连接是否正常
- 尝试使用其他 IP 源
- 检查 IP 更新脚本是否存在

### 3. 浏览器无法打开怎么办？
- 检查 Chrome 浏览器是否正确安装
- 检查代理工具是否正常运行
- 查看命令行窗口的错误信息
